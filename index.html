Guía de Instalación
PARTE 1 — Publicar en GitHub Pages (desde la PC)
1
Crear cuenta en GitHub
Ir a github.com → Sign up → completar con email, contraseña y nombre de usuario (ej: mantto-fi-unne). Confirmar el email.
2
Crear un repositorio nuevo
Una vez dentro de GitHub → botón verde "New" → completar:
• Repository name: ot-mantenimiento
• Marcar Public
• Tildar ✅ "Add a README file"
→ Clic en Create repository
3
Subir el archivo index.html
Dentro del repositorio → clic en "Add file" → "Upload files" → arrastrar el archivo index.html descargado de esta app → clic en "Commit changes".
⚠ El archivo DEBE llamarse exactamente index.html (en minúsculas)
4
Activar GitHub Pages
En el repositorio → Settings (engranaje) → sección Pages en el menú izquierdo → Source: Deploy from a branch → Branch: main → carpeta: / (root) → clic Save.

Esperar 1-2 minutos. La URL de la app será:
https://TU-USUARIO.github.io/ot-mantenimiento/
✓ Esta URL funciona desde cualquier dispositivo con internet. Es gratuita y permanente.
5
Verificar que funciona
Abrir la URL en el navegador. Deberías ver la app con el header naranja de UNNE. Si ves un error 404, esperar 2 minutos más y recargar.
PARTE 2 — Instalar en la Tablet Android
1
Abrir en Chrome
En la tablet, abrir Google Chrome (no Samsung Internet, no Firefox). Escribir la URL de GitHub Pages en la barra de direcciones.
⚠ Debe ser Chrome para Android. Otros navegadores no permiten instalar la app.
2
Instalar como app
Una vez que cargue la app, Chrome mostrará automáticamente un banner en la parte inferior que dice "Agregar a pantalla de inicio". Tocar ese banner.

Si no aparece el banner: tocar los 3 puntos (⋮) arriba a la derecha → "Agregar a pantalla de inicio" → confirmar.
3
Confirmar instalación
Aparece un diálogo que pregunta el nombre. Dejarlo como "Mantenimiento UNNE" → tocar Agregar.

El ícono 🔧 aparece en la pantalla de inicio de la tablet. Al abrirla, se abre en pantalla completa como una app nativa, sin barras del navegador.
4
Funciona sin internet
Los datos se guardan localmente en la tablet. Se puede trabajar sin conexión. Cuando haya wifi, ir a ⚙ Config → Subir pendientes a Sheets para sincronizar con Google Sheets.
✓ Repetir los pasos 1-3 en la PC del escritorio para tener la app también ahí.
PARTE 3 — Conectar con Google Sheets (opcional pero recomendado)
1
Abrir Gestion_OT_2026 en Sheets
En Google Drive → abrir el archivo Gestion_OT_2026.
2
Abrir Apps Script
Menú Extensiones → Apps Script. Se abre una nueva pestaña con el editor.
3
Pegar el código
En el editor, borrar todo lo que hay y pegar el siguiente código completo:
// ══════════════════════════════════════════ // APPS SCRIPT — Mantenimiento UNNE // Pegar en Extensions > Apps Script de Gestion_OT_2026 // ══════════════════════════════════════════ const SPREADSHEET_ID = '1uXPU666muqP3-5X-BuYBbRcy2MSf5u471lvJDcm68n4'; const SHEET_OT = 'Registro OT'; const DATA_START = 3; function doGet(e) { const a = e.parameter.action || ''; if (a === 'ping') return res({ ok: true, msg: 'Script activo OK' }); if (a === 'getOTs') return getOTs(); return res({ ok: false, msg: 'Accion no reconocida' }); } function doPost(e) { try { const b = JSON.parse(e.postData.contents); if (b.action === 'appendOT') return appendOT(b.data); if (b.action === 'updateEstado') return updateEstado(b.nroOT, b.estado, b.fecha, b.hora); return res({ ok: false, msg: 'Accion desconocida' }); } catch(err) { return res({ ok: false, msg: err.message }); } } function appendOT(row) { const ss = SpreadsheetApp.openById(SPREADSHEET_ID); const sh = ss.getSheetByName(SHEET_OT); if (!sh) return res({ ok: false, msg: 'Pestaña no encontrada: ' + SHEET_OT }); const data = sh.getDataRange().getValues(); const nros = data.slice(DATA_START-1).map(r => r[0]).filter(Boolean); if (nros.includes(row[0])) { for (let i = DATA_START-1; i < data.length; i++) { if (data[i][0] === row[0]) { sh.getRange(i+1, 1, 1, row.length).setValues([row]); return res({ ok: true, msg: 'OT actualizada: ' + row[0] }); } } } const last = data.slice(DATA_START-1).reduce((acc,r,i) => r[0] ? DATA_START+i : acc, DATA_START-1); sh.getRange(last+1, 1, 1, row.length).setValues([row]); const col = { abierta:'#e8f4fd', en_proceso:'#fef9e7', cerrada:'#eafaf1' }; if (col[row[17]]) sh.getRange(last+1,1,1,row.length).setBackground(col[row[17]]); sh.getRange(last+1,1).setFontWeight('bold'); return res({ ok: true, msg: 'OT registrada: ' + row[0] }); } function updateEstado(nro, estado, fecha, hora) { const ss = SpreadsheetApp.openById(SPREADSHEET_ID); const sh = ss.getSheetByName(SHEET_OT); const data = sh.getDataRange().getValues(); for (let i = DATA_START-1; i < data.length; i++) { if (data[i][0] === nro) { sh.getRange(i+1,16).setValue(fecha||''); sh.getRange(i+1,17).setValue(hora||''); sh.getRange(i+1,18).setValue(estado); return res({ ok: true, msg: 'Estado actualizado' }); } } return res({ ok: false, msg: 'OT no encontrada: ' + nro }); } function getOTs() { const ss = SpreadsheetApp.openById(SPREADSHEET_ID); const sh = ss.getSheetByName(SHEET_OT); const data = sh.getDataRange().getValues(); const ots = data.slice(DATA_START-1).filter(r => r[0]); return res({ ok: true, ots: ots, count: ots.length }); } function res(data) { return ContentService.createTextOutput(JSON.stringify(data)) .setMimeType(ContentService.MimeType.JSON); }
4
Guardar y publicar
1. Guardar con Ctrl+S
2. Clic en Implementar (azul, arriba a la derecha) → Nueva implementación
3. Tipo: Aplicación web
4. Ejecutar como: Yo
5. Quién tiene acceso: Cualquier persona
6. Clic en Implementar
7. Autorizar cuando pida permisos
8. Copiar la URL que aparece (empieza con https://script.google.com...)
⚠ Guardar esa URL. Es la que va en ⚙ Config → URL del Web App.
5
Pegar la URL en la app
Ir a ⚙ Config en esta app → pegar la URL en el campo → clic Guardar → clic Probar conexión. Debe aparecer "Script activo OK".
✓ Desde ese momento, cada OT que se registre se guarda automáticamente en tu planilla de Google Sheets.
Preguntas frecuentes
¿Se pierden los datos si no hay internet?
No. Todo se guarda localmente en el dispositivo. Al recuperar conexión, hacer sync manual desde ⚙ Config.
¿Cómo actualizo la app cuando haya cambios?
Subir el nuevo index.html al repositorio de GitHub (mismo proceso que el paso 3 de la Parte 1). GitHub Pages se actualiza solo en 1-2 minutos.
¿Funciona en la PC del escritorio también?
Sí. Abrir la URL en Chrome de la PC y hacer el mismo proceso de "Agregar a pantalla de inicio" o instalar como app desde los 3 puntos del navegador.
¿La app se actualiza en todos los dispositivos al mismo tiempo?
Los datos locales son independientes por dispositivo. Google Sheets actúa como base central: cada dispositivo puede subir (↑) y traer (↓) datos desde Sheets.
¿Tiene costo?
Cero. GitHub Pages es gratis para repositorios públicos. Google Apps Script es gratis. El único requisito es una cuenta de Google y una cuenta de GitHub.
